# Smart Campus Navigation & Event Hub

A modern mobile application designed to simplify campus life for students, offering intuitive navigation and a centralized hub for events and schedules.

## Key Features

*   **Secure Authentication:** Seamless login experience with Email/Password or Google Sign-in, including email verification and password reset functionalities.
*   **Interactive Campus Map:** A dynamic 2D map with smart navigation powered by the A* AI algorithm, providing optimal pathfinding between campus locations.
*   **Real-time Event & Lecture Hub:** A comprehensive calendar displaying real-time events, lectures, and schedules. Admins have the capability to add new events.
*   **Instant Push Notifications:** Stay updated with timely alerts and announcements.
*   **User-Friendly Interface:** Supports both Light & Dark Modes and offers multi-language support (English & Arabic) with RTL layout adjustments.
*   **Robust Technology Stack:** Built with Flutter for a smooth, cross-platform user experience and Firebase for a scalable and reliable backend.

## Detailed Features from Demo Analysis

### 1. Splash Screen

A visually appealing splash screen with a dark blue background, a glowing green location pin icon, and the app's title "Smart Campus" with the tagline "Navigate Your Campus Experience."

### 2. Interactive Campus Map

*   **Search Bar:** Prominent "Where do you want to go?" search functionality.
*   **2D Map Layout:** Detailed, color-coded floor plans of various buildings (e.g., Mainstream Building, Credit Building) and specific rooms (Lab 001, Lab 004, Lab A).
*   **Navigation Pathfinding:** Clearly drawn blue paths indicating the shortest route between selected start and end points.
*   **Map Controls:** Intuitive controls for zooming, locating the user's current position, and floor selection.
*   **Bottom Navigation Bar:** Persistent navigation to Map, Calendar, and Settings screens.

### 3. Event Hub (Calendar Screen)

*   **Categorized View:** Tabs for "Events" and "Lectures."
*   **Detailed Listings:** Each item displays date, time, title (e.g., "Mobile Computing," "Math 6"), a brief description, and a location link.
*   **Admin Functionality:** A floating "+" button for authorized users to add new events with fields for Event Type, Title, Description, Location, and Date/Time.

### 4. User Settings & Profile

*   **Profile Management:** Displays user's name, email, and profile picture with an "Edit Profile" option.
*   **Customizable Profile:** Users can update their username, select their Grade (Junior, Senior, Graduate, Faculty, Staff), and Department (Computer Science, Engineering, Business).
*   **Appearance Settings:** Toggle switch for seamless Light/Dark Mode switching.
*   **Language Settings:** Support for English and Arabic, with automatic RTL layout adjustments for Arabic.
*   **Security Options:** "Change Password" and "Logout" functionalities.

### 5. Authentication Flow

*   **Login Screen:** Standard email/password fields, "Remember me" option, and "Forgot Password?" link.
*   **Social Login:** Integration with "Sign in with Google."
*   **Registration Screen:** Multi-field form for new user setup (Photo, Username, Email, Grade, Department, Password).
*   **Email Verification:** A dedicated screen for email verification, followed by a "Proceed to Login" confirmation.
*   **Forgot Password:** Simple interface to request a password reset link.

## Technologies Used

*   **Flutter:** UI Toolkit for building natively compiled applications for mobile, web, and desktop from a single codebase.
*   **Firebase:** Backend-as-a-Service (BaaS) providing authentication, real-time database, and other cloud services.
*   **A* AI Algorithm:** For efficient and smart pathfinding on the campus map.
